# Objectif général

Réaliser une application Android composée de deux onglets :

- un pour convertir les températures (°C ↔ °F)

- un autre pour convertir les distances (Km ↔ Miles)

L’application doit aussi :

- afficher un menu “Quitter”

- intercepter la touche Retour pour demander confirmation avant de fermer.


# Test et comportement attendu
Lancer l’application.

Onglet “Température” : saisir 25 → Résultat 77°F.

Onglet “Distance” : saisir 10 → Résultat 6.21 miles.

Appuyer sur Retour → boîte de dialogue “Voulez-vous vraiment quitter ?”.
