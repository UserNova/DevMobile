# Aperçu :

Ce que tu construis : un écran Formulaire (saisie Nom, E-mail, Téléphone, Adresse, Ville) ➜ un écran Récapitulatif (affichage des données).

Compétences : layouts XML, EditText, Button, Intent explicite, putExtra/getStringExtra, finish(), validations de base.

# Pré-requis : 

Android Studio (n’importe quelle version récente)

Projet Empty Activity (Java), Min SDK 24+

# Test pas à pas : 

- Lancer l’app sur l’émulateur.

- Saisir Nom + Email (+ autres champs).

- Cliquer « Envoyer » → l’écran 2 s’ouvre et affiche le récap.

- Cliquer « Retour » → on revient au formulaire.
